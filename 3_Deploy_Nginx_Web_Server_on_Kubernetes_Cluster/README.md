# Deploy Nginx Web Server on Kubernetes Cluster

## Overview
This section covers the deployment of Nginx web server on a Kubernetes cluster. Learn how to containerize, deploy, and manage Nginx services.

## Topics Covered
- Creating Nginx container images
- Kubernetes Deployment manifests
- Service configuration
- Ingress setup
- Load balancing
- SSL/TLS termination

## Key Objectives
- Deploy Nginx pods on Kubernetes
- Expose Nginx service internally and externally
- Configure Ingress for HTTP(S) routing
- Scale Nginx deployments
- Implement health checks

## Deployment Steps
1. Build or pull Nginx container image
2. Create Deployment manifest
3. Create Service for pod exposure
4. Configure Ingress (optional)
5. Verify and test deployment
